# MovieDB-Android

Android app demo using The Movie Database (TMDb) API.

# Features:

1. Movie list Screen
 - Favorite list
 - Pagination
 - Pull to refresh
 - Search by title and year in the list

2. Movie details Screen
- Movie summary 
- Movie trailers
- Scene transition 

# Structure and libraries
- MVP architecture
- Material Design
- Retrofit to consume API
- Glide to download images
